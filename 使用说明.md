# 📋 使用说明（无依赖版本）

## 此版本特点
✅ 不需要安装任何额外依赖（无需 npm install vuex）
✅ 直接用 HBuilderX 新建项目后覆盖文件即可运行

## 操作步骤

### 第一步：HBuilderX 新建项目
1. 文件 → 新建 → 项目
2. 选择「uni-app」，Vue版本「Vue3」，模板「默认模板」

### 第二步：覆盖文件
把本压缩包所有文件按路径覆盖到新项目：

  App.vue              → 项目根目录
  main.js              → 项目根目录
  pages.json           → 项目根目录
  manifest.json        → 项目根目录
  uni.scss             → 项目根目录
  store/index.js       → 新建 store 文件夹放入
  pages/task/task.vue      → pages/task/
  pages/recipe/recipe.vue  → pages/recipe/
  pages/plan/plan.vue      → pages/plan/

### 第三步：直接运行
菜单 → 运行 → 运行到小程序模拟器 → 微信开发者工具
无需 npm install！

## 自定义信息
打开 store/index.js，修改顶部 coupleInfo：
  name1/name2    → 你们的昵称
  startDate      → 在一起的纪念日
